To Do List Application

For my favorite program, I took the To Do List project that I learned in Java Programming and added a database to it. I realized when the user closes the program, it doesn't save the to-do text that was entered. That bothered me, so I used SQLite database to help keep the data saved. Now, when you open and close it, the program saves the to-do text that was entered.

Here is a sample of the code from my project.


            while (resultSet.next()) {
              String text = resultSet.getString("to_do");
              toDo tDL = new toDo(text);
              getAll.add(tDL);
          }
          return getAll;
